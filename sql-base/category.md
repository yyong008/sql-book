# 特性分类

根据 SQL 不同的特性，可以将 SQL 分为一下四个种类：

## DDL

>Data Definition Language，数据定义语言

- CREATE
- ALTER
- DROP
- TRUNCATE

## DML

>Data Manipulation Language，数据操作语言

- SELECT
- INSERT
- UPDATE
- DELETE

## DCL

>Data Control Language，数据控制语言 

- GRANT
- REVOKE

## TCL

>Transaction Control Language，事务控制语言

- COMMIT
- ROLLBACK
- SAVEPOINT
