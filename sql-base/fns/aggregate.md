# 聚合函数

| 函数             | 描述                                       | 使用类型                             |
|------------------|--------------------------------------------|--------------------------------------|
| `SUM()`          | 计算总和                                   | 通用                                 |
| `AVG()`          | 计算平均值                                 | 通用                                 |
| `COUNT()`        | 计算行数或非NULL值的数量                    | 通用                                 |
| `MAX()`          | 返回最大值                                 | 通用                                 |
| `MIN()`          | 返回最小值                                 | 通用                                 |
