# 特定数据库函数

| 函数             | 描述                                       | 使用类型                             |
|------------------|--------------------------------------------|--------------------------------------|
| `TO_CHAR()`      | 将日期/时间转换为字符串，特定于 PostgreSQL | PostgreSQL                           |
| `DATE_FORMAT()`  | 将日期/时间转换为字符串，特定于 MySQL       | MySQL                                |
