# 字符串函数

| 函数             | 描述                                       | 使用类型                             |
|------------------|--------------------------------------------|--------------------------------------|
| `CONCAT()`       | 连接两个或多个字符串                         | 通用                                 |
| `SUBSTRING()`    | 提取子字符串                               | 通用                                 |
| `UPPER()`        | 将字符串转换为大写                         | 通用                                 |
| `LOWER()`        | 将字符串转换为小写                         | 通用                                 |
| `LENGTH()`       | 返回字符串的长度                           | 通用                                 |
| `TRIM()`         | 去除字符串两侧的空格或指定字符             | 通用                                 |
| `REPLACE()`      | 替换字符串中的子字符串                     | 通用                                 |
