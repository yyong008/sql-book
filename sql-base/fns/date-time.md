# 日期和时间函数

| 函数             | 描述                                       | 使用类型                             |
|------------------|--------------------------------------------|--------------------------------------|
| `NOW()`          | 返回当前日期和时间                         | 通用                                 |
| `DATE()`         | 从日期/时间表达式中提取日期部分            | 通用                                 |
| `TIME()`         | 从日期/时间表达式中提取时间部分            | 通用                                 |
| `YEAR()`         | 从日期中提取年份                           | 通用                                 |
| `MONTH()`        | 从日期中提取月份                           | 通用                                 |
