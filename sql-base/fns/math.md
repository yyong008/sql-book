# 数学函数

| 函数             | 描述                                       | 使用类型                             |
|------------------|--------------------------------------------|--------------------------------------|
| `ROUND()`        | 四舍五入到指定的小数位数                   | 通用                                 |
| `ABS()`          | 返回绝对值                                 | 通用                                 |
| `CEIL()`         | 向上取整                                   | 通用                                 |
| `FLOOR()`        | 向下取整                                   | 通用                                 |
| `POWER()`        | 返回一个数的指定次幂                       | 通用                                 |
