# 类型转换函数

| 函数             | 描述                                       | 使用类型                             |
|------------------|--------------------------------------------|--------------------------------------|
| `CAST()`         | 将一个类型转换为另一个类型                 | 通用                                 |
| `CONVERT()`      | 将一个类型转换为另一个类型，特定于数据库   | 特定于数据库                           |
