# DROP

| 对象             | 描述                                                       |
|------------------|------------------------------------------------------------|
| `DROP TABLE`     | 删除数据库中的表格及其所有数据。                             |
| `DROP INDEX`     | 删除数据库中的索引。                                       |
| `DROP VIEW`      | 删除数据库中的视图。                                       |
| `DROP TRIGGER`   | 删除数据库中的触发器。                                     |
| `DROP DATABASE`  | 删除整个数据库及其所有相关对象。                           |
