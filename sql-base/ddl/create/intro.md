# create

| 类型           | 描述                                       |
|----------------|--------------------------------------------|
| `CREATE TABLE` | 用于创建新表格，定义表的结构和约束条件。     |
| `CREATE INDEX` | 在表的一个或多个列上创建索引，以提高查询性能。 |
| `CREATE VIEW`  | 创建虚拟表，基于一个或多个查询的结果集构建。   |
| `CREATE TRIGGER` | 创建触发器，当特定事件发生时自动执行一系列的 SQL 语句。 |
